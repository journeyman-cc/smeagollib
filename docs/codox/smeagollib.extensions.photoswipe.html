process-simple-slide

(process-simple-slide slide)

Process a single slide, as decoded by simple-grammar. At this stage a slide is expected to be represented as a sequence of vectors, one for each property of the slide (:title, :src). Each vector contains the name of the property as a keyword as its first element, and the value of the property as its second element.

Returns a map of these properties, with, if possible, :w (width), :h (height), and :msrc (source URL of a low resolution variant) added.
