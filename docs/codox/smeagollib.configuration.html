transform-map

(transform-map m tuples)

transform this map m by applying these transforms. Each transforms is expected to comprise a map with the keys :from and :to, whose values are respectively a key to match and a key to replace that match with, and optionally a key :transform, whose value is a function of one argument to be used to transform the value of that key.
