diff

(diff git-directory-path file-path version)(diff git-directory-path file-path older newer)

Find the diff in the file at file-path within the repository at git-directory-path between versions older and newer or between the specified version and the current version of the file. Returns the diff as a string.

Based on JGit Cookbook ShowFileDiff.

fetch-version

(fetch-version git-directory-path file-path version)

Return (as a String) the text of this version of the file at this file-path in the git directory at this git-directory-path.

Based on JGit Cookbook ReadFileFromCommit.

load-or-init-repo

(load-or-init-repo git-directory-path)

There’s a bootstrapping problem: when Smeagol first starts the repository hasn’t been initialised. Try to open the repository at this git-directory-path; if an exception is thrown, try to init a repository at this git-directory-path, and then open it and populate it.
