smeagollib.util

Miscellaneous utility functions supporting Smeagol.

content-dir

The absolute path to the directory in which Wiki content (i.e., Markdown files) are stored.

get-message

(get-message message-key request)(get-message message-key default request)

Return the message with this message-key from this request. if not found, return this default, if provided; else return the message-key.

get-messages

get-servlet-context-path

(get-servlet-context-path request)

not-servable-reason

(not-servable-reason file-path)

As a string, the reason this file-path cannot safely be served, or nil if it is safe to serve. This reason may be logged, but should not be shown to remote users, as it would allow file system probing.
