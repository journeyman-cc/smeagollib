smeagollib.authenticate

Authentication functions.

*password-file-path*

dynamic

Path to the password file. Dynamic, primarily to allow it to be overridden for test purposes.

add-user

(add-user username newpass email admin)

Add a user to the passwd file with this username, initial password newpass, email address and admin flag; or, modify an existing user. Return true if user is successfully stored, false otherwise.

view source

authenticate

(authenticate username password)

Return true if this username/password pair match, false otherwise

view source

change-pass

(change-pass username oldpass newpass)

Change the password for the user with this username and oldpass to this newpass. Return true if password was successfully changed. Subsequent to user change, their password will be encrypted.

(delete-user username)

Delete the user with this username from the password file.

evaluate-password

(evaluate-password pass1 pass2)(evaluate-password password)

Evaluate whether this proposed password is suitable for use; return true is so, a keyword if not.
